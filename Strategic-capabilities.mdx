## Strategy Execution

<CodeGroup>
```typescript Simple Strategy
// Basic momentum strategy
const strategy = await alina.createStrategy({
  type: 'momentum',
  timeframe: '4h',
  risk_level: 'moderate'
});
```

```typescript Complex Strategy
// Multi-factor strategy
const strategy = await alina.createStrategy({
  type: 'custom',
  factors: ['sentiment', 'volume', 'social_activity'],
  conditions: {
    sentiment: '> 70',
    volume_change: '> 200%',
    social_score: '> 85'
  }
});
```
</CodeGroup>
