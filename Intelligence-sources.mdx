## Integration Methods

<CodeGroup>
  ```typescript Custom Source
  // Add custom data source
  await alina.addDataSource({
    type: 'telegram',
    channel: 'crypto_signals',
    priority: 'high'
  });
  ```

  ```typescript Data Query
  // Query processed data
  const sentiment = await alina.getData({
    source: 'twitter',
    topic: 'DeFi',
    timeframe: '24h'
  });
  ```
</CodeGroup>
